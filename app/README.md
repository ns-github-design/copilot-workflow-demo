# TrailMix Campus Demo App

A lightweight student travel planning demo designed for your "Scaling Development Teams with Copilots" session.

## Run locally

Option 1 (VS Code Live Server):
- Open `app/index.html`
- Start Live Server

Option 2 (Python):
- From repo root: `cd app`
- Run: `python -m http.server 5500`
- Open: `http://localhost:5500`

## Demo flow for students

1. Click **Load Demo Trip**.
2. Show generated itinerary and student-focused metrics.
3. Change destination/group size/budget style/trip vibe.
4. Click **Generate Student Copilot Plan** and explain how AI accelerates student collaboration.
